# AI Productivity Assistant

A Python AI assistant powered by OpenAI.

## Features

- ✉️ Write professional emails
- 🌍 Translate text
- 📄 Summarize articles
- 🐍 Explain Python code
- 💾 Save results to text files

## Technologies

- Python 3
- OpenAI API
- python-dotenv

## Installation

```bash
pip install -r requirements.txt
```

## Run

```bash
python app.py
```

## Environment Variables

Create a `.env` file:

```text
OPENAI_API_KEY=your_api_key_here
```
